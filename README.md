# Mavis SDK examples

These are the SDK examples which include

- [Mavis Market SDK example](https://github.com/axieinfinity/mavis-marketplace-sdk)
- [Tanto Connect kit example](https://github.com/skymavis/tanto-kit)


## Requirements

React 18 or later

## Setup

```bash
yarn setup
# or
npm setup
```

## Run project

```bash
npm run dev
# or
yarn dev
```
